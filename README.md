Este es el proyecto SAKU: Abyss Echoing Through, se verá acomoañado de su
información técnica gracias al documento Word GDD (Game Design Document),
con el cual se justifica su existencia y se hablan otros detalles sobre
el juego, exclusivos para el ojo público.

Con este texto leído, usted acepta este proyecto como un adicional de esfuerzo
de tres tontuelos queriendo hacer un juego decente en Godot, con la esperanza
de tener un GOTY pronto, y admite que este juego, a pesar de sus fallas, logró
cautivarlo con su historia y personajes.

-Axel
